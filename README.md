# T2_kubernetes_project

# IMDB Pipeline
This project is a Kubernetes-based pipeline for predicting IMDB scores using machine learning models. It includes a user interface for data input and model prediction, as well as a training module for updating the model with new data.

# Features
- User interface for inputting movie data and predicting IMDB scores.
- Model training module that allows users to train the model with new data.
- Integration with a machine learning model for score prediction.
- Dockerized application for easy deployment.
- Kubernetes configuration for scaling and managing the application.
